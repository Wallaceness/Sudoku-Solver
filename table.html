Sudoku Solver

Instructions:

This solver will provide a valid solution to any solvable sudoku that is put into it. To use it, just enter all the numbers provided for the puzzle in the appropriate boxes of the table below and hit the solve button at the bottom. If the arrangement of numbers is unsolvable, the program will return "Invalid sudoku"; otherwise, it will give a solution. The program can also generate new puzzles. To do that, just enter the quantity of filled-in boxes you want the puzzle to start with and hit the generate button. The program will randomly select that many numbers and return the result into the table. Please note that many sudokus, particularly those that have around 20 starting numbers or less, have more than one valid solution (someone should tell that to sudoku.com...). That's it; I hope you like the solver and feel free to e-mail me at ncw232@gmail.com if you experience any difficulties.
